# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2025-07-10

### Added
- 🎮 Complete 3D virtual pet system with Three.js
- 🐾 12 unique species with hand-crafted ASCII art (Phoenix, Dragon, Neko, Wolf, Slime, Owl, Unicorn, Robot, Ghost, Turtle, Bunny, Fox)
- 🎨 Canvas-based 3D ASCII rendering with glow/shadow effects
- ✨ Post-processing pipeline (Bloom + Vignette)
- 💬 AI-powered pet chat using LLM (z-ai-web-dev-sdk)
- 🎮 6 interactive actions (Feed, Play, Train, Heal, Rest, Pet)
- 📊 Token tracking system for session analytics
- 🎲 Deterministic pet generation (Mulberry32 PRNG)
- 🏆 XP and leveling system (50 XP per level)
- 😊 Mood system (Ecstatic, Happy, Neutral, Sad, Critical)
- 🖥️ Dark cyberpunk UI with 1px border radius
- 📱 Responsive 3-column layout
- 🔄 Framer Motion animations throughout
- 🗄️ SQLite persistence with Prisma ORM
- 📡 REST API (species, pets, actions, chat)
- 🏅 5 rarity levels (Common to Legendary)
- 30 floating 3D particles with mood-matching colors
- Dynamic background and lighting based on pet mood
- Session stats dashboard (tokens, actions, messages)
- Professional GitHub repository setup
