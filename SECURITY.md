# Security Policy

## 📢 Reporting Security Vulnerabilities

We take security seriously. If you discover a security vulnerability, please report it responsibly.

### How to Report

**Do NOT** open a public issue for security vulnerabilities.

Instead, please send an email to the repository owner or use GitHub's [private vulnerability reporting](https://github.com/smouj/asciicritters/security/advisories/new).

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response Timeline

- **Acknowledgment**: Within 48 hours
- **Assessment**: Within 7 days
- **Resolution**: As soon as possible, with priority

### Supported Versions

| Version | Supported |
|---------|-----------|
| Latest main branch | ✅ Yes |
| Older versions | ❌ No |

Thank you for helping keep ASCIICritters safe! 🛡️
